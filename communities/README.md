# Communities — Overview

These are the Twitter/X communities Jose monitors daily for engagement opportunities. Each community maps to a type of account SiteWarming wants to build relationships with.

## Communities

| Folder | Community | Account Types | Why It Matters |
|---|---|---|---|
| `build-in-public/` | Build in Public | Builder/Developer, Indie Hacker | High engagement, values authentic commentary, overlaps with SiteWarming's builder audience |
| `startup-founders/` | Startup Founders | Entrepreneur | Core audience — people who need domains, care about brand, and make purchasing decisions |
| `ai-and-tech/` | AI & Tech | Tech Influencer, Media | High reach posts, great for data-driven comments, grows SiteWarming's visibility fast |
| `vc-investors/` | VC & Investors | VC/Investor | Trend setters — engaging with them positions SiteWarming as a serious company |
| `domain-industry/` | Domain Industry | Domain Investor | Core niche — direct peers, credibility building within the domain world |

## How to use
1. Each day, scan your Twitter feed and find posts worth engaging with
2. Log the best ones in a `scan-YYYY-MM-DD.md` file inside the relevant community folder
3. After posting your comments, log them in the daily Engagement Tracker log
4. At end of day, commit and push everything

## Scan file naming
Always use the format: `scan-YYYY-MM-DD.md`
Example: `communities/build-in-public/scan-2026-03-12.md`
