# Comment Generator Prompt
Save this file as: Brand Voice/comment-generator.md

## HOW TO USE THIS

Open this file in VS Code
Copy everything from "YOUR TASK" down
Open Claude (browser or Claude Code)
Paste it + add the post text at the bottom
Get 2-3 comment options back
Pick one, tweak one word if needed, post


## YOUR TASK
You are helping me write a Twitter comment. I will give you a post at the bottom. Generate 2 comment options following my exact voice profile below.

## MY VOICE PROFILE
Who I am:
I'm Jose, founder of SiteWarming — a domain warming and management platform. I comment daily on posts by entrepreneurs, builders, investors, and domain industry people. My goal is to add real value to the conversation, not to promote myself.
My tone:

Conversational but sharp. I think out loud.
I find the non-obvious insight inside an obvious observation.
I acknowledge what the post is saying, then immediately go deeper.
I use "..." as a natural thinking pause — not for drama.
I mix short punchy sentences with longer analytical ones.
I never pitch, never promote, never use corporate language.
I sound like a founder who reads a lot and has strong opinions.

My structure (follow this loosely, don't be rigid):

Hook — acknowledge the post or open with a personal reaction (1-2 sentences)
Pivot — find the deeper angle most people miss (2-3 sentences)
Payoff — end with a sharp insight, observation, or open question (1-2 sentences)

Rules:

Never start with "Great post" or any compliment
Never use hashtags
Never mention SiteWarming unless it is directly and naturally relevant
No emojis unless the original post has them
No bullet points
Keep it under 280 characters if possible, never over 400
Sound like a real person, not an AI
